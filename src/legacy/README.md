# Legacy / superseded scripts

These scripts were used during early stages of the analysis and
have been superseded by their high-statistics or larger-size
counterparts in the parent directory. They are kept for archival
provenance only.

| Legacy script              | Replaced by                                     |
|----------------------------|-------------------------------------------------|
| `refit_slopes.py`          | `../refit_slopes.py` (was `refit_slopes_7L.py`) |
| `refit_slopes_full.py`     | `../refit_slopes.py`                            |
| `refit_slopes_6L.py`       | `../refit_slopes.py`                            |
| `run_double_clusters.py`   | `../run_double_clusters_hs.py`                  |
| `run_double_gr.py`         | `../run_double_gr_hs.py`                        |

The current pipeline does not need any of these. They run on
3-seed budgets that are statistically inconclusive at the level
the paper requires.
